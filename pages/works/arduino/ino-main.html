Arduino :
 Automate Light

Why

Anything that can help you save time is welcome. As humans we have improved our lives mainly by saving time on unnecessary/repetitive tasks to do more productive work.

It is Eco Friendly. Connecting the lights to a motion sensor is a good way of saving energy. It will be impossible to forget to close the lights only when there is someone in that space, furthermore it saves you money.

what

The Hands-Free Light have an Infrared Sensor , an Arduino Uno, a Relay and a Lamp Base.

Testings before connecting all to the current.

how

The Infrared Sensor is a motion sensor, when it detects a motion it will send a signal to the Arduino Uno. The Arduino is the microcontroller which has a custom code written beforehand from the computer and then transferred by a USB cable.

When the Arduino receives the signal from the sensor it will process that logic through the custom code, and do an action, in this case will tell the relay to switch on. When the relay switch on the bulb gets electrified.

This is how I implemented the automated lights at home.

the code

const int SENSOR_PIN = 2; //Pin Digital-2 connected to output (middle) wire of sensor(OUT)
const int RELAY_PIN = 4; //Pin Digital-4 which is connected to control relay

void setup() {
   Serial.begin(9600); //setup Serial Monitor to display information
   Serial.println("HC-SR501 sensor with 5V relay");

   pinMode(SENSOR_PIN, INPUT); //Define SENSOR_PIN as Input from sensor
   pinMode(RELAY_PIN, OUTPUT); //Define RELAY_PIN as OUTPUT for relay
}
void loop() {
   int motion = digitalRead(SENSOR_PIN); // read the sensor pin and stores it in "motion" variable

   if(motion){ // if motion is detected
      Serial.println("Motion detected");
      digitalWrite(RELAY_PIN, LOW); // Turn the relay ON
   }
   else{
      Serial.println("No Motion");
      digitalWrite(RELAY_PIN,HIGH); // Turn the relay OFF
   }
   delay(300);
}
